How to run the program

Use this file, index.html.(tested on chrome)

Or install NodeJS, run node parser2.js, be sure datas.json is in same directory.

About

Source code is in coffee, compiled js and helping html file is enclosed so that run the code is no more than installing a modern browser.

datas.json contains test datas I used, it's in plain text format and can be read & modified with any editor.

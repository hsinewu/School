readme
基本上client可以不開，直接telnet localhost 54321即可
實作了存款(d)、提款(w)、轉帳(t)、查詢(i)
ex
d 30
w 20
t 50 usr2

登入時先輸入帳號再輸入密碼
如果是未記錄的帳號，輸入的就是新密碼
可以正常轉帳到還沒有登入過的人
初始時只定義一個帳號tony密碼123戶頭有100
要求輸入帳號時右邊括號內的數字表示還能嘗試幾次登入

server那邊的視窗為了方便觀察做了個可以察看帳戶和存款的功能
輸入bank或acc查看
只要server沒有重開這些資訊就不會消失

實在太難搞

姑且只把拉出來的介面留著
